## 🔹 Qwen Code কী?

**Qwen Code** বলতে বোঝায় Qwen AI model-এর সেই অংশ/ব্যবহার, যেটা দিয়ে:

* কোড লেখা যায়
* কোড explain করা যায়
* bug fix করা যায়
* full project generate করা যায়
* CLI / API দিয়ে automation করা যায়

মানে, এটা basically **AI-powered coding assistant**, যেমন:

* GitHub Copilot
* OpenAI Codex
* Cursor IDE
