# ⚠️ Important Reality (এটা জানতেই হবে)

Antigravity powerful, কিন্তু risky:

👉 AI automatically command run করতে পারে
👉 ভুল হলে problem হতে পারে

📌 real issue:

* ভুল command দিলে file delete হতে পারে
* security risk থাকতে পারে ([TechRadar][5])

👉 তাই blindly trust করা যাবে না
